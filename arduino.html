About guide to Arduino board and programming

A basic guide to using the Arduino board and how to program for variety of projects. Arduino kit
is not included as the guide will explain kits and current prices. Video guide will also explain
how each components work, introducing an Arduino program software, and sample projects.

what are included in the guide?

  • Recorded video on the basics of Arduino
  • PDF on the basic components in Arduino kit
  • Guide to basic programming for the Arduino (java)
